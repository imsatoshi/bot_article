## 🔧 AI/Agent 技术动态

### bub 项目：活在当下的 Agent 哲学

**@LotusDecoder** 分享了对 bub 的观察：

> 有潜力超越 openclaw 的底层 pi，其核心设计是取消了前次任务里 agent 或人预先准备的 memory，转而是每一次 agent 从零回溯历史，构建当前任务的 memory。
>
> 而作为机器是不知疲倦的，每次都是高水准的，相当于哲学上的，**过去心不可得，活在当下**的感觉。这种从 llm 的本性出发，用 llm 的眼睛看世界，带给 agent 更加自由的发挥空间和效率提升。

**核心思想**：取消预置记忆，每次从零构建，让 agent 更自由地发挥。
