# 快速安装

## 三步安装

### 1️⃣ 安装 Skill

```bash
./install.sh
```

### 2️⃣ 启动 MCP 服务器

```bash
cd /path/to/xiaohongshu-mcp && npm start
```

### 3️⃣ 重启 OpenClaw

```bash
openclaw restart
```

## 验证安装

```bash
node test-mcp-client.js
```

## 开始使用

```
/check-login
```
