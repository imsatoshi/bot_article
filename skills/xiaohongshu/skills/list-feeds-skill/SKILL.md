# List Feeds Skill

Get the list of feeds from Xiaohongshu homepage using the xiaohongshu-mcp HTTP API.

## Execution

### Run the script

```bash
node skills/list-feeds-skill/scripts/list-feeds.mjs
```

### Return the result

After execution, return the list of feeds to the caller.
